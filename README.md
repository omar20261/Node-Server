# ========= Docker setup =========

## prerequisite :
  Docker  
  Docker-compose

## runing & building commends :
  docker-compose up
  
note: you need to add sudo at the beginning of your os is linux or mac

 ## navigate to http://localhost:3500


# ========= manual setup =========

## prerequisite :
  nodejs  
  npm  
  mongodb  

## runing commends :
  npm i   
  npm start
  
 ## navigate to http://localhost:3500

# ========= Testing =========
